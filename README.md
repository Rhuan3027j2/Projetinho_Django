# Projeto Exemplo 01 - Django

## Como rodar o projeto
1. Criar ambiente virtual: `python -m venv venv`
2. Ativar venv: `venv\Scripts\activate` (Windows) ou `source venv/bin/activate` (Linux/Mac)
3. Instalar Django: `pip install django`
4. Rodar o servidor: `python manage.py runserver`

## Tecnologias e Banco de Dados
- **Linguagem:** Python
- **Framework:** Django
- **Banco de Dados:** SQLite (padrão do Django)
- **Ambiente:** venv (Virtual Environment)
